# Personal Portfolio Website

This is my personal portfolio website hosted on GitHub Pages. You can visit it at [https://nrjsingh1.github.io](https://nrjsingh1.github.io).

## Features

- Responsive design
- Clean and modern UI
- Sections for About, Projects, and Contact
- Social media links
- Skills showcase

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- GitHub Pages for hosting

## Local Development

To run this website locally:

1. Clone the repository
2. Open `index.html` in your browser
3. Make changes as needed
